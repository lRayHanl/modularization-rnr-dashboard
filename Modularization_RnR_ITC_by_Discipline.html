Note 3. 견적 단계에서는 Sea Fastening/Grillage Q'ty Fabrication Scope으로 반영, 수행 단계에서 Grillage는 Barge 제원이 결정된 후 Optimize 될 수 있음.
Note 4. 견적 단계에서 Module 업체 미 선정 시, Proposal Stage의 1. Feasibility Study 및 2. Engineering Work for Proposal을 포함 수행한다.
Note 5. Project 특성에 따라 On-shore/Off-shore 분리 계약인 경우, 필요에 따라 Module 계약을 공사에서 구매 Item으로 전환하여 수행할 수 있음. (Ex. 캐나다 Blackgold Project 경우 Local Regulation, Tax 등으로 구매 Item으로 전환함)
Note 6. Barge Spec.을 확정하여 제공, Grillage Scope 결정, Barge Arrangement, 운항 횟수를 담고 있는 Batch Plan을 작성 (Grillage Beam 결정 인자)
Note 7. 지역적 특성, Transportation 가격 경쟁력을 고려하여 GS 물류에서 수행할 수 있음.
